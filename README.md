# 💊GESTION-PHARMACIE

## 📌Description
Ce projet est une application console de gestion de pharmacie développée en Java. Il permet de gérer efficacement les médicaments, les clients, les ordonnances ainsi que les ventes.
L'objectif est de faciliter le travail du pharmacien en automatisant les tâches courantes.

## Fonctionnalités
- Gestion des médicaments
    - Ajouter un médicament
    - Modifier un médicament
    - Supprimer un médicament
    - Gestion du stock
    - Médicaments avec ou sans ordonnance
    - Suivi des dates (expiration, renouvellement)
- Gestion des clients
    - Enregistrement des clients
    - Historique des achats
- Gestion des ordonnances
    - Vérification de la validité
    - Gestion des durées (1 mois, 6 mois,...)
- Gestion des ventes 
    - Vente avec ou sans ordonnance
    - Calcul automatique du total
